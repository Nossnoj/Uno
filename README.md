Detta är en konsolversion av kortspelet UNO. I denna versionen så spelas spelet med totalt fyra spelare (1 mänsklig och 3 datorstyrda spelare).

Spelet följer de klassiska UNO-reglerna:

- Matcha färg eller symbol med det översta kortet på bordet.
- Specialkort som +2, +4, Reverse och Wild ändrar spelets gång.
- Den som först blir av med alla sina kort vinner.
- Vid 1 kort kvar så ska UNO ropas!

Enda regeln som inte är implementerad är att kedja flera kort av samma symbol. Det går t.ex inte att lägga flera fyror på varandra.

Hur man spelar:
- Skriv en indexsiffra för att spela ett kort (t.ex 3 för att spela det tredje kortet på handen).
- Skriv "d" för att dra ett kort från högen.
- Vid Wild kort så får du välja färg vilket du gör genom att skriva in en färg som input.
- När spelaren har 1 kort kvar så kommer det att kunna gå att ange "uno" som input och ifall man missar det så får man plocka två kort.
